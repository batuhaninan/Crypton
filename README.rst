================
From Pip
================


**You can install pycrypton from pip**

.. code-block:: console

    $ pip install pycrypton


|
|
|

================
From Source
================

You need Python 3.7+ to use Crypton

|

*********************
Install Dependencies
*********************

.. code-block:: console

    $ pip install .

|


***************************************************************
Install Dev Dependencies (for running tests)
***************************************************************

.. code-block:: console

    $ pip install -r requirements_dev.txt

|

*********************
Run Tests
*********************

.. code-block:: console

    $ pytest
